# 🏠 RentNest - Property Rental Platform

A modern, full-stack property rental platform built with Next.js, featuring real-time reviews, property listings, and a seamless user experience.

![RentNest Banner](https://via.placeholder.com/1200x400/4F46E5/FFFFFF?text=RentNest)

## ✨ Features

### 🏢 Property Management
- **Property Listings**: Browse available properties with detailed information
- **Advanced Search**: Filter properties by location, price, type, and amenities
- **Property Details**: View comprehensive property information with high-quality images
- **Favorites**: Save favorite properties for later viewing

### ⭐ Customer Reviews
- **Real-time Reviews**: View and submit property reviews
- **Rating System**: 5-star rating system with visual feedback
- **User Photos**: Profile pictures with graceful fallbacks
- **Verified Reviews**: Authentic reviews from verified tenants

### 🎨 User Experience
- **Responsive Design**: Fully responsive across all devices
- **Dark Mode**: Built-in dark mode support
- **Smooth Animations**: Framer Motion animations for delightful interactions
- **Loading States**: Beautiful loading animations and skeleton screens
- **Error Handling**: Graceful error states with user-friendly messages

### 🔐 Authentication & Security
- Secure user authentication
- Protected routes and API endpoints
- Role-based access control

## 🚀 Tech Stack

### Frontend
- **Framework**: [Next.js 14](https://nextjs.org/) (App Router)
- **Language**: JavaScript/React
- **Styling**: Tailwind CSS with custom components
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Icons**: Lucide React & React Icons
- **State Management**: React Hooks (useState, useEffect)
- **HTTP Client**: Native Fetch API

### Backend
- **API Routes**: Next.js API Routes
- **Database**: MongoDB with Mongoose ODM
- **Authentication**: NextAuth.js
- **Validation**: Custom validation with error handling

### Development Tools
- **Package Manager**: npm/yarn
- **Version Control**: Git
- **Code Quality**: ESLint, Prettier
